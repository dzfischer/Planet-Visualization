Our Data

Below is a searchable table of all the data we used in our various charts.

The data is sourced from the NASA Exoplanet Archive, which is the most up to date database available of all known planets.

We grabbed a CSV of the data and uploaded it into our Heroku cloud-hosted PostgreSQL (upload done manually with TablePlus). We then reflect the data into memory using Flask-SQLAlchemy, create a json-ified route to access it via Flask, load it below via Jinja, and make into a pretty table using Bootstrap and the 'DataTables' JS library. The below DataTable library has a feature which allows us to pull in the data via JSON.
