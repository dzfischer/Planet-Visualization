Exoplanet Discovery & Visualizations

Group Project 2

Brief Project Summary

NASA has a fairly extensive database of exoplanets & requisite data about them. We thought this would be an ideal data set to use in our visualizations.

We used the below libraries to take this data set and create some charts to explore it with. We also thought it would be neat to create a searchable table of all this data, which can be found by following the "Explore" link to the left
